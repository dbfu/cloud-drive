1.1 项目背景

随着云计算技术的发展，个人云盘已经成为日常工作中不可或缺的工具。本项目致力于提供一个简单、实用的云盘解决方案。

# 项目笔记

## 开发计划

### 第一阶段：基础架构
- 设置项目结构
- 配置开发环境
- 实现用户认证系统

### 第二阶段：核心功能
- 文件上传和下载
- 文件列表管理
- 文件夹操作

### 第三阶段：增强功能
- 文件预览功能
- 搜索和筛选
- 性能优化

## 技术要点

### 前端
- 使用 Zustand 进行状态管理
- Tailwind CSS 实现响应式设计
- 组件化开发，保持代码整洁

### 后端
- Fastify 高性能框架
- Prisma ORM 简化数据库操作
- JWT 认证保护 API

### 文件处理
- 支持多种文件格式
- 文件大小限制 100MB
- 用户隔离存储

## 注意事项

1. 确保所有 API 都有认证保护
2. 文件名需要唯一化处理
3. 定期清理临时文件
4. 监控存储空间使用情况
